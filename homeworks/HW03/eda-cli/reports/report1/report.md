# Отчет о example.csv

Исходный файл: `example.csv`

Строк: **36**, столбцов: **14**

## Качество данных (эвристики)

- Оценка качества: **0.64**
- Макс. доля пропусков по колонке: **5.56%**
- Дублированных строк: **1**
- Есть столбцы с полностью одинаковыми значениями: **False**
- Слишком мало строк: **True**
- Слишком много колонок: **False**
- Слишком много пропусков: **False**

Детализация оценки качества (**0.64**).
Начальное значение **score: 1.0**. Качество данных снижается, если:
- Присутствуют пропуски - вычитаем долю пропусков из score
- Есть дублирующиеся строки - вычитаем 0.1 за каждую такую строку
- Есть столбец или несколько столбцов, в которых все значения одинаковые - вычитаем 0.1
- Слишком мало строк (меньше 100) - вычитаем 0.2
- Слишком много столбцов (больше 100) - вычитаем 0.1
