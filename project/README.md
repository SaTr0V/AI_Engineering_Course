- **Project name:** `UpscaleCompare: A Comparative Study of Image Super-Resolution Techniques`
- **Author:** `Petrov Alexander Yurievich`

- **Contact:** `@SaTr0V`

**Brief description:**  
> This project is dedicated to building a web service for image upscaling using various methods - from simple mathematical approaches to neural networks.
> Open datasets DIV2K, FFHQ, and FLICKR2K (mentioned) are used.
> The result is a web service that offers users several images enhanced by different models.

## 2. Project structure

The project is organized with the following structure:

- `requirements.txt` – project dependencies.
- `report.md` - project report (problem statement, data, experiments, results).
- `report_rus.md` - project report in Russian.
- `self-checklist.md` – self-check checklist before submission.
- `notebooks/` – experimental notebooks:
  - EDA, model prototypes, individual experiments.
- `src/` – main project codeа:
  - modules for data processing;
  - model training and inference;
  - service launcher.
- `data/` – demo/training data:
  - **only open, synthetic, or anonymized data**;
  - small samples for quick startup.
- `configs/` – configuration files:
  - `.yaml`, `.json`, `.env.example`, etc.
- `tests/` – tests (unit tests, simple checks).
- `artifacts/` – saved models, reports, experiment results.

For convenience, every README is available in both Russian and English (chekout `README_RUS.md`).
